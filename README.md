# АлисЁноК — Эстетика природы в вашем доме

Современный, адаптивный одностраничный сайт для бренда товаров для дома из натуральных материалов (джут, водный гиацинт).

## Стек технологий
- **HTML5** (Семантическая верстка)
- **Tailwind CSS** (Стилизация)
- **Vanilla JavaScript** (Анимации и интерактивность)
- **Google Fonts** (Playfair Display & Montserrat)

## Структура проекта
- `index.html` — главная страница
- `style.css` — пользовательские стили
- `script.js` — логика анимаций и меню
- `/images` — папка с изображениями (используйте `input_file_0.png` - `input_file_7.png`)

## Как запустить на GitHub Pages
1. Создайте новый репозиторий на GitHub.
2. Загрузите все файлы проекта в основную ветку (`main`).
3. Перейдите в **Settings** > **Pages**.
4. В разделе **Build and deployment** выберите ветку `main` и нажмите **Save**.
5. Через пару минут ваш сайт будет доступен по адресу `https://ваш-логин.github.io/название-репозитория/`.
